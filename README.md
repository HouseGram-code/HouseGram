## üöÄ Getting Started

### Prerequisites
*   Node.js (v16+)
*   npm or yarn

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/yourusername/housegram.git
    cd housegram
    ```

2.  **Install dependencies**
    ```bash
    npm install
    # or
    yarn install
    ```

3.  **Run the app**
    ```bash
    npm start
    ```

4.  **Open in Browser**
    Visit `http://localhost:3000` to start chatting!
