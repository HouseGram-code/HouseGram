## üöÄ Getting Started

Follow these steps to get HouseGram running on your local machine.

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/yourusername/housegram.git
    cd housegram
    ```

2.  **Install dependencies**
    ```bash
    npm install
    # or
    yarn install
    ```

3.  **Configure Firebase**
    - Create a project in the [Firebase Console](https://console.firebase.google.com/).
    - Enable **Authentication** (Email/Password).
    - Enable **Firestore Database**.
    - Enable **Storage**.
    - Add your configuration keys to the project.

4.  **Run the app**
    ```bash
    npm start
    ```

5.  **Open in Browser**
    Visit `http://localhost:3000` to start chatting!

Go to **Settings > My Profile**. Tap your avatar to upload a photo or a short video (up to 5s). You can also update your bio and name instantly.
